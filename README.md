# User-Workstation-Virtualization-Lab
This project simulates a real-world end-user workstation in a virtualized environment. The goal of this lab is to demonstrate foundational IT support skills such as operating system installation, user account management, network connectivity verification, and troubleshooting common workstation issues.

This lab reflects tasks commonly performed by IT Support Specialists and Field Technicians during new workstation deployments and break/fix support.
